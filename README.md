# Job Application Tracker (Django + MySQL)

A full‑stack web application that helps users track job applications, interviews, and companies.  
Built using **Django**, **MySQL**, **HTML**, and **CSS**.

---

## 🚀 Features

- Multi‑table CRUD (Companies, Jobs, Applications, Interviews)
- One‑to‑Many relationships (Company → Jobs → Applications → Interviews)
- Transaction logic (Offer acceptance creates an interview log)
- Server‑side validation
- Summary dashboard with SQL aggregates
- Clean HTML/CSS frontend
- Fully normalized database (3rd Normal Form)

---

## 🛠 Tech Stack

- **Backend:** Django 5
- **Database:** MySQL Workbench
- **Frontend:** HTML5, CSS3
- **ORM:** Django Models
- **Version Control:** Git + GitHub
